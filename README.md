Primeiro Projeto: Analise de Numeros -> https://onlinegdb.com/89mXOENb9

Esse projeto de inicio pega quantos numeros o cliente deseja digitar, logo em seguida é pedido para o cliente digitar os números ( somente inteiros ),
após a digitação dos números o sistema pegara eles e armazena em variaveis, no qual te devolve, a soma de todos os numeros pares, a quantidade dos números,
impares e o maior númeor digitado pelo cliente. Caso o cliente digite 0 na quantidade de nùmeros que deseja digtar, o programa retornara : a soma dos pares 0,
a quantidade de numeros impares 0 e o maior número 0.
Nesse programa foi utilizado da iA chatgpt, assim como informações de outra fonte escolar e adaptações vinda de mim. Foi utilizada 
como principal linguagem o java, por eu ter mais conhecimento, e como hospedador foi utilizado o site : onlinegbd, a plataforma mais simples que é de meu conhecimento.
De inicio é pedido para que o cliente digite a quantidade de números por causa da liberdade que o cliente ganha, ao invés de limitar ele, entreguei liberdade
para que ele decidice melhor o resultado final, já que as vezes você pode querer saber mais do que só 3 ou 5 números.
Para rodar esse programa será necessário somente entrar no link do começo, não é necessário intalar nenhuma dependencia, claro além de um navegador.

infelizmente não foi feito uso de commits já que sou iniciante nessa parte não detinha conhecimento para o uso.

Segundo projeto: CRUD web -> https://www.mediafire.com/folder/58flbupwptchv/teste_junior_dev

Esse projeto é feito para pegar nome/titulo de qualquel coisa que o cliente deseje, de inicio pede para digitar o titulo no qual deve ter 3 caracteres ou mais,
caso não tenha o programa retornara uma mensagem dizendo "Título deve ter no mínimo 3 caracteres", a descrição é opcional, e como padrão o status é todo, pode escolher outros
como doing ou done, o id não é necessário colocar pois o banco já o cria sozinho, e mesmo que coloque algum id ele não cadastrara.
Agora os comando, é possivel criar como já dito, é possivel editar os dados, somente buscar o id que deseja que logo em seguida aparecera as informções nos campos,
logo você pode mudar oque quiser ( menos o id ), clicando em atualizar muda os dados criados.
o buscar id assim como dito você escolhera um id e chama os dados desse id fazendo ele ser mostrado nos campos, alem disso caso não saiba seu id,
existe uma tabela logo abaixo que atualiza toda vez que é criado um novo registro, essa lista pode ser filtrada por status, então é só procurar.
temos o delete agora que caso você não tenha gostado ou queira só sumir com seu registro, selecione seu id e clique no botão excluir.
as tecnologias usadas foram a IA chatgpt, IA gemini, IA deepseek, assim como meu conhecimento, tambem foi usado a plataforma vscode.
O projeto é inicializado no chrome mesmo, mas para o funcinamento é necessário baixar a pasta que esta no link do mediafire (localizado acima dessa descrição), descompactala,
abrir ela e rodar um programa, sendo ele o USBWebserver, que ira rodar o banco de dados local para o funcionamento do programa,
além disse é necessário abrir o site via comando no chrome, comando:Localhost:8080 , isso ira abrir um site que você deve entrar em teste-dev e em seguida html_teste.html, para que o site funcione.
As principais linguagens utilizadas foram: html, java script, css e php.
